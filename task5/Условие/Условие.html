Саша ведет бюджет и анализирует, как изменялся баланс на его счету. Он выписал числа ﻿a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​﻿ — изменения баланса за последние ﻿nnn﻿ дней.

Отрезок из дней ﻿[i,j][i,j][i,j]﻿ Саша считает разумным, если суммарное изменение баланса с ﻿iii﻿-го по ﻿jjj﻿-й день равно нулю, т.е. ﻿ai+ai+1+⋯+aj=0a_i + a_{i+1} + \dots + a_j=0ai​+ai+1​+⋯+aj​=0﻿. Отрезок из дней ﻿[l,r][l,r][l,r]﻿ считается нормальным, если внутри данного отрезка можно найти разумный подотрезок.

Помогите Саше проанализировать эффективность ведения бюджета и посчитайте количество нормальных отрезков в массиве изменений баланса его счета.

Формат входных данных

Первая строка содержит число ﻿nnn﻿ ﻿(1≤n≤2⋅105)(1 \leq n \leq 2 \cdot 10^5)(1≤n≤2⋅105)﻿ — количество дней, для которых Саша записывал изменения баланса. Вторая строка содержит ﻿nn n﻿ чисел ﻿a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​﻿ ﻿(−109≤ai≤109)(-10^9 \leq a_i \leq 10^9)(−109≤ai​≤109)﻿ — значения изменений баланса.

Формат выходных данных

Выведите одно число — количество нормальных отрезков для данного массива.
